Dictionaries

CodeStacks - Python

Abstract

Dictionaries in Python exist so that you can refer to values in hashed form so you don't need to type in long strings repeatedly. You can create them in many ways and you can also use them in many ways.

Overview

To create a dictionary, start with a variable. Then set the value of that variable to a pair of curly brackets like so: 

dict = {}

Now that we have a dictionary created, let's add some keys to store: 

keys = {
    "name": "Alice",
    "age": 30,
    "job": ["Python", "Data"]
}

As seen above, you can use arrays to store multiple values inside a single key. Of course, it not as ethical, but it's still something you can do.

Accessing Values

Now that you can create dictionaries and add keys to hash inside said dictionaries, let's look at how to access the data that you store. 

keys = {
    "name": "Alice",
    "age": 30    
}

To access values, you need to call the name of the dictionary, and then the name of the key inside square brackets.
You can also access them with the name.get() method, with the name of the key inside. 

keys = {
    "name": "Alice",
    "age": 30
}

print(f"{keys['name']} is {keys['age']} years old.")

In the above example, we take the values of the dictionary, keys, and we print the values of it, "name" and "age" in to a sentence using f" strings.

Injecting Values

Python dictionaries, like arrays, are dynamic, so you can very easily add new values to them. Instead of adding the index, you can add the name of the key you want to add, and you set a value.

In order to inject values, you need to specify the content and the content of the item you want to inject.

You can do this with insert() or append(). 

# Injecting Values:
keys = {
    "name": "Alice", 
    "age": 30
}

# Inserting a value:
# First value: Key name,
# Second value: Content;
keys["job"] = "Python"

# Checking the updated dictionary:
for i in keys:
    print(keys[i])

Output: 

Alice
30
Python

Removing Values

If you need to remove values from dictionaries, you can use the del method or the name.pop() method. You can apply it to a dictionary like so: 

# Example:
keys = {
    "name": "Alice",
    "age": 30,
    "city": "New York",
    "job": "Python"
}

for i in keys:
    print(keys[i])

del keys["age"]
delete = keys.pop("city")

for j in keys:
    print([j])

So essentially, we create a dictionary, then we delete two values, age, and "city". So now, the output should look like this if you run it: 

Alice
