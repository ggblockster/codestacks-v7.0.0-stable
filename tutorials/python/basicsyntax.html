Basic Syntax

CodeStacks - Python

Abstract

When coding in Python, there's a key syntax you need to follow. This syntax is like the language your device understands. Python as a coding language was developed then released in 1991 as an English look-alike to make it easy to learn, making it one of the first languages any programmer jumps to when beginning their journey.

Overview

When coding in Python, you need to understand syntax. Operations need to be shown as certain symbols, and they follow the order of operations for calculations, (which will be covered later here). Brackets are extremely useful when you need to separate items for the order of operations, covered here.

String Syntax

This cannot be overstated enough, and it is REALLY important for how you type them. When you use quotations in Python, it must be only one type for a string or another. Those are the single and double mark, they are interchangeable. The example below shows just how they should be used when you have to deal with strings: 

# Valid Usage
print("Hello, Python!")
print('Hello, Python!')

# Invalid Usage
print("Hello, Python!')
print('Hello, Python!")

... notice the mismatch in quotation marks in the invalid code? Mismatches in quotations will cause Python to throw an error, so it will fail to run the rest of the code so watch out for that!

Useful Functions

Some really useful functions you'll find yourself using a lot on Command Prompt output code is the print() command, and the input() command. You'll also find yourself using str() and int(), but not as much as print() and input().

The way you'd put them in code basically look like this: 

# Print Command:
print("Hello, Python!")

# Input Command:
_ = input("Enter some text: ")

The PRINT Command

As shown above on how to use it, the print command does one thing: it takes any string (text) that it finds within quotations and prints it to the output. Like this: 

Hello, Python!

Inputs

This is also what you would see in your command prompt when you use an input() command — except for the fact that you have to type text in. A really important factor is that an input prompt must be assigned to a variable. More on variables will be covered over later. Below, is an example of locking the auto-closing part of the terminal, using the input prompt: 

# The Input Command:
text = input("What's your name?")

print("Hello, ", text, "!")

The code above simply prints whatever you typed.

Integers and Floats

Integers in Python are whole numbers, positive or negative. You can convert different values into integers and back, however, it can only a number, more specifically one without place value smaller than 1.

If you want a number with decimals, then floats are what you're looking for. To convert something to an integer, you need to wrap it inside int(). Similarly, to convert something to a float value, you use float().


Here's an example:

# Convert a string to an integer:
string = "123"
integer = int(string)

print(integer)

...and if you wanted to convert them to floats instead: 

# Convert a string to a float:
string = "12.34"
float = float(string)

print(float)

Oh, and by the way, you can print integers and floats, but it's easier to manage as strings.

String Manipulation

Strings are defined with quotations. To change them into an integer or float, you can use the int() or float() commands respectively. Since we covered how to convert strings to integers or floats, the next step is converting values to strings. All values are compatible with strings, but not the other way. 

# Convert integers to strings:
int = 123
str = str(int) # Returns "123", as a string.
