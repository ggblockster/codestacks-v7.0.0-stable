Arrays

CodeStacks - Python

Abstract

Arrays in Python are basically variables that can hold multiple pieces of data at once, and those values can be called all from the same variable. To access them, you need to call the variable and the item's index.

Overview

To create an array, let's start by defining a variable: 

array = ""

Now that we have a variable, let's add some data inside it: 

array = ["stick", "potion", "wand"]

To define that a variable has multiple values, you need to place the values inside a pair of square brackets, as shown above.

In order to place multiple values, you need to separate them by commas. In the above example, only strings were used, but you can use any data type inside arrays.

Accessing Values

Now that you know how to create arrays, let's look at how to access the values that you put inside the array.

Each value in arrays are indexed, starting at a value of 0. That means that the first value has an index of 0, the second being 1, and so on. To access these values, you call the variable name, and then in square brackets, you place the item index. For example, if we wanted to grab "potion" from the array above, we would have to structure it like so: 

# Example:
array = ["stick", "potion", "wand"]

# Printing out the value:
print(array[1])

"potion", in this case, has an index value of 1, so if you call the index value of 1, the array will return "potion".

Injecting Values

Python arrays are dynamic, so if you need to inject values into them at runtime, you can.

In order to inject values, you need to specify the content and the index of the item you want to inject.

You can do this with insert() or append(). 

# Injecting Values:
array = [1, 3, 4, 5]

# Inserting a value:
# First value: Index,
# Second value: Item;
array.insert(1, 2)

# Appending a value to the end:
array.append(6)

for i in array:
    print(i)

Output: 

1
2
3
4
5
6

Removing Values

If you need to remove values from arrays, you can use the remove() method. You can apply it to an array like so: 

# Example:
array = ["stick", "potion", "wand"]

array.remove("stick")

for i in array:
    print(i)

In the above snippet, we define an array with the items stick, potion, and wand. Next, we remove stick by calling remove(). The output should look like this: 

potion
wand
