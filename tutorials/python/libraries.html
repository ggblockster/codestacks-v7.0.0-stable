Libraries

CodeStacks - Python

Abstract

Libraries in Python are an easy way to access features that are not built in to normal Python. Examples are random and time, both of which you cannot use without importing them into your Python file first.

Overview

To import a library, you use the import keyword. From that, you add the name of the library you wish to import. In the below example, let's import the random library: 

import random

But if you only need a specific part of a library without importing the whole thing, you can do this instead, and as an example, let's use time. 

# Time
from time import sleep

In this instance, we are importing the sleep module from the time library.

Random Numbers

Since random numbers cannot be generated in Basic Python, you can import the corresponding library, random like so: 

# Example:
import random

Now that we have access to the random module, we can now use random.randint() to generate the number. Because we are using randint(), you need to provide a starting integer, and an ending integer for the range in which you want to generate the random number from: 

# Random Integer
import random

num = random.randint(3, 8)
print(num)

In the code snippet above, we define two variables to define the start and end of the range for the random integer. Then, we call random.randint() and input the two variables. We then print the end result to the output window.

Time

If you need to add a time delay to your code, like for a countdown, you need the time library. But this time, you only need to import the sleep module. In order import only the sleep module from the time library, you can do this: 

# Importing sleep()
from time import sleep

Now that we have access to the sleep() command, we can add any number inside it, and it will pause the runtime code for the specified amount of seconds. 

# Example Usage:
from time import sleep

name = input("What's your name? ")
sleep(1)
print(f"Hello, {name}!")

In the snippet above, we store some user input in a variable, name. Then, we wait for 1 second before printing Hello, and then the value of the name variable.

So if you typed in something like "Bob", the output would look like this: 

Hello, Bob!

OS and SYS

There are also libraries for managing what's displayed on the terminal that your code is outputting to. Those libraries are os and sys.

To use them, you need to add the following to the top of your code (technically, it doesn't matter where it is, but it's good for readability). 

# Importing OS and SYS:
import os
import sys

Now, we can impact the terminal window itself, not just the text displayed on it.


Clearing the Screen

In order to wipe the text that is currently display on the terminal, you can do it with os.
The thing is, there's two ways to do it, and it depends on the operating system that you do it on.

For Windows, you do it like this: 

import os
os.system("cls")

But on every other operating system, like MacOS, Linux, Unix, etc, you need to do this instead: 

import os
os.system("clear")

But if you need to run your code on any platform but you don't know what operating system someone might be running it on, you can import the platform library. 

import os
import platform

if platform.system().lower() == "windows":
    os.system("cls")
else:
    os.system("clear")

What the code above does is it checks if the operating system, converted to lowercase is equal to windows. If it is, then it runs os.system("cls"), but if it isn't, then it runs os.system("clear") instead.
