Enums

CodeStacks - Python

Abstract

Like dictionaries, you can access values of enums with a key so you can easily organise your data in one place of your code. The difference, is that enums aren't built in to Python but are from a library, enum.

Overview

In order to create an enum, first, let's import the Enum module from the enum library. 

from enum import enum

Now that the module has been imported, let's create an enum. In Python, to create enums, you need to create a class with Enum as the parameter. 

from enum import Enum

class season(Enum):
    SPRING = 1
    SUMMER = 2
    AUTUMN = 3
    WINTER = 4

Accessing Values

Now that you have an enum set up, let's look at how you access the values inside the enum. To access the value, you need to refer to the name of the enum, then the name of the value, then an optional property of it that you want to access. 

# Example:
from enum import Enum

class Season(Enum):
    SPRING = 1
    SUMMER = 2
    AUTUMN = 3
    WINTER = 4

print(Season.SPRING.name) # "SPRING"
print(Season.AUTUMN.value) # 3
print(type(Season.SPRING)) # <enum>
print(list(Season)) # All in a list.

And the output: 

SPRING
3
<enum 'Season'>
[<Season.SPRING: 1>, <Season.SUMMER: 2>, 
<Season.AUTUMN: 3>, <Season.WINTER: 4>]

So the properties are name and value, which return the name and value you set for the key you specify.

Then, there's type() and list(). Inside type(), you add the key you want the details from, and Python will output the relative source code. For list(), you only have to specify the name of the enum and Python will output the details of every key in that enum.

Iterations

You can also use loops to display all the values of enumerators.
In order to do that, you need to first define the enum, then the name of the enum inside a loop, replacing i with something more fitting. However, you also might need the value of i itself, so you can tag .value at the end, so that Python will output the index of the key instead of the value it holds. 

# Iterating over Values:
from enum import Enum

class Season(Enum):
    SPRING = 1
    SUMMER = 2
    AUTUMN = 3
    WINTER = 4

for season in Season:
    print(f"{season.value} - {season.name}.")

Output: 

1 - SPRING.
2 - SUMMER.
3 - AUTUMN.
4 - WINTER.
