Colours

CodeStacks - Python

Abstract

These aren't really fundamental toward coding in Python, but it's more or less a "nice to know". You can customise your terminal with coloured text, or colouring the terminal itself. Let's go over how.

Overview

In Python, in the terminal, you can display text in not just white, but any hexadecimal colour. You can also customise the terminal window, but it can only be one of 16 colours.

Coloured Text

In order to even start to render your text in colour, you need to add the following snippet to your code: 

def hex_to_rgb(hex_colour):
#Convert a hex color code to an RGB tuple.
    hex_colour = hex_colour.lstrip('#')
    colour = tuple(int(hex_color[i:i+2]
    return colour, 16) for i in (0, 2, 4))

def colored_text(text, hex_colour):
# Return ANSI escape code for the colour.
    r, g, b = hex_to_rgb(hex_colour)
    rgb = "{r};{g};{b}m{text}"
    return f"\033[38;2;rgb\033[0m"

Now that this snippet is a part of your code, you can add coloured text by calling the colored_text() command inside print().

Here's an example of some coloured text: 

# Example
def hex_to_rgb(hex_color):
    """Convert a hex color code to an RGB tuple."""
    hex_color = hex_color.lstrip('#')
    return tuple(int(hex_color[i:i+2], 16) for i in (0, 2, 4))

def colored_text(text, hex_color):
    """Return text formatted with a specific hex color using ANSI escape codes."""
    r, g, b = hex_to_rgb(hex_color)
    return f"\033[38;2;{r};{g};{b}m{text}\033[0m"

print(colored_text(f"This text is red!", "#FF6145"))

_ = input()

You can literally paste that in to any code editor as a Python file, but it will only work if you run it with native Python, and not any specific Python online compiler.

Coloured Terminals


Windows

On top of simply colouring the text in the terminal, you can colour the literal terminal itself! The way you do it involves the os library.

On Windows, you need to call os.system(), and then you add color, then the background and text colours that you want the console to display. Here's an example: 

# Custom Background, Windows:
import os

os.system("color B0")

Notice how in the snippet above, it says B0? Well, in Windows Python, you can't use colour names, but colour codes. The background colour goes first, and then the text colour. Below is the full colour chart that this uses:

But wait a minute — weren't we using hex colour codes just a minute ago? Well, to answer that question, we need to look back to how Windows handled its colours. It had two ways of showing colours, light, and bright (8 and F), and it couldn't parse RGB value strings.



MacOS, Linux, Unix

On macOS, you can't natively change the background colour of the terminal using code, rather, you have to change it on your device settings. And the same pretty much goes for Linux and macOS.
