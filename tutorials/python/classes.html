Classes

CodeStacks - Python

Abstract

Classes in Python are like blueprints. They allow for more object-oriented programming so that you can create more structured code.

Overview

To create a class, you need to use the class keyword, and then the name of the class, followed by a pair of brackets and a colon.

For now, we'll add temporary values to the class so we can modify it later. 

class myClass():
    name = ""
    age = 0
    job = ""

Adding Values

Now that you have a class created, let's add some values to make it usuable. When you create the class, you can add initial values that you can use straight away. 

# Example:
class Season():
    SPRING = 1
    SUMMER = 2
    AUTUMN = 3
    WINTER = 4

But if you want to add values later on, you need to create an object of a class. Then, you can modify its values. 

# Example
class Bike:
    name = ""
    gear = ""

# Creating an object.
bike = Bike()

bike.name = "BMX"
bike.gear = 21

print(f"Bike name: {bike.name}, Gear: {bike.gear}")

So we have a class named Bike, and we create a variable holding the object data. We set the name of the bike to "BMX" and the gear property to 21. So the output should look like this: 

Bike name: BMX, Gear: 21

Deleting Objects

Let's say you need to delete an object that you created inside a class, through code. The way you would do it is very similar to the method with dictionaries — the del keyword. 

# Deleting Objects:
class myClass():
    name = "Alice"
    age = 0

name = myClass()
age = myClass()
age.value = 30

# Deleting an object:
del age

print(myClass.name, myClass.age)

The above code will now produce an error because there is no object in the class with the name, age. That is because it has been deleted.

The PASS Statement

Classes in Python are not supposed to be empty, but if for some reason you may want to have an empty class, you can add the pass statement inside, which will effectively prevent the code from returning an error. 

class myClass():
    pass
